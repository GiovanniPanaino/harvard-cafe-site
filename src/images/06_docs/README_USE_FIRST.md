# Harvard Café Website Starter Image Pack

Prepared as a **starter / placeholder image pack** for a single-page Harvard Café / Rand Airport website concept.

## What is inside

- `00_web_ready/` — compressed `.webp` files ready for website mockups and early builds.
- `01_hero/` to `05_functions_placeholders/` — original downloaded source files from Creative Commons / public-domain sources.
- `06_docs/IMAGE_SOURCE_MANIFEST.csv` — source, author and licence notes for each image.
- `06_docs/REPLACEMENT_SHOT_LIST.md` — recommended photo list to capture from Harvard Café before final launch.

## Important legal note

This pack avoids using scraped Harvard Café photos from Facebook, Instagram, TripAdvisor, RestaurantGuru or Google because those images may be copyrighted by the restaurant, photographers or review platforms.

For final launch, replace placeholders with:

1. Official Harvard Café logo files.
2. Current high-resolution food photos from the restaurant.
3. Current interior/exterior/apron photos taken with permission.
4. Current menu artwork supplied by the owners.
5. Airshow/function photos cleared by the photographer/organiser.

## Licence warning

Some Rand Airport / aircraft images are **Creative Commons Attribution-ShareAlike**. That generally allows use, including commercial use, but you must give credit, link to the licence and indicate changes. ShareAlike may affect edited/remixed versions.

For the lowest-risk final commercial website, use this pack for mockups only and replace with owner-supplied material.
